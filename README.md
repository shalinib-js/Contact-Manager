# Contact-Manager
A simple, responsive web app to store and manage contact information.
Built with beginner-friendly HTML, CSS, and JavaScript.

## Features

- Add new contacts (name, email, phone number)
- Edit contact details inline
- Delete individual contacts
- Save all contacts in localStorage
- Add a profile image/icon for each contact
- Clean and responsive UI using HTML, CSS, and JavaScript
